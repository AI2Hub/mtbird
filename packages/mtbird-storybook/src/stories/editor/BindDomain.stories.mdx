# 绑定域名

域名是绑定到应用上的，每个应用只可以绑定一个域名。

## 设置域名

进入后台，选择目标应用，点击设置，找到 `绑定域名`

<img src={BindDomain1} />

## 添加证书

绑定域名需填写域名（如 test.staringos.com 无协议和路径）和 [Https 证书信息](https://www.runoob.com/w3cnote/https-ssl-intro.html)。证书可以使用常用的证书服务商，可以查看这个[文档申请免费证书](https://zhuanlan.zhihu.com/p/174755007)

<img src={BindDomain2} />

## 切换 DNS 解析

切换域名的 DNS 解析 需要您通过您购买域名的域名服务商处理，这里列几个常用的 服务商

- [阿里云 添加网站解析](https://www.alibabacloud.com/help/zh/alibaba-cloud-dns/latest/add-an-a-record-to-a-website-domain)
- [腾讯云 添加网站解析](https://cloud.tencent.com/document/product/302/3446)

需增加一条域名解析，请设置您的域名 DNS 解析类型为 cname，解析值为文本框链接
