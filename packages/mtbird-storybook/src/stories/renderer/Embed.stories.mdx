# 嵌入渲染器

渲染器是经过编辑器编辑的页面渲染的 React 组件，可以无缝嵌入到任意 React 系统中，也可以通过如 乾坤 等微前端框架嵌入到 Vue 或 React 项目中。

## 安装渲染器

```shell
yarn add @mtbird/renderer @mtbird/shared
```

## 引用渲染器

```tsx
import React from 'react'
import {IPageConfig} from '@mtbird/shared'

interface IProps {
  pageConfig: IPageConfig;
}

const RenderPage = ({pageConfig}: IProps) => {
  const innerDataSource = mew InnerDataSource();
  return <Renderer pageConfig={pageConfig} dataSource={innerDataSource} />;
};
```
