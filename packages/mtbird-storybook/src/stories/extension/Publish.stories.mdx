# 发布

打开 Mtbird 后台，切换到对应的团队。

<img src={Publish1} />

点击开发者菜单->获取 token

<img src={Publish2} />

<img src={Publish3} />

复制 token，打开拓展代码，在代码根目录执行

```shell
NODE_ENV=production mtbird publish --token=[token]
[token]替换为在后台获取的 Token
```

最后，执行发布命令，`publish` 命令会首先执行 `build` 对拓展代码进行编译，随后会进行代码上传。

常见错误

1. [Error] Unauthorized
   Token 已过期，或者生成 Token 的团队选择不对，不是创建拓展的团队
2. [Error] 拓展版本已存在，请更新版本
   拓展版本已存在，请在 manifest.json 中更新版本号
