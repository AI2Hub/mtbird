# 拓展

拓展系统是星搭精卫的核心功能，帮助不同级别的客户和开发者，根据自己的业务需要拓展增强编辑器、渲染器功能。

拓展包括三个组件的组成部分：

- <LinkTo kind="拓展-contribute-编辑器贡献--page">编辑器贡献 (Contribute)</LinkTo>: 主要针对增强编辑器功能的增强。
- <LinkTo kind="拓展-pipe-渲染管线--page">渲染管线 (Pipe)</LinkTo>: 增强渲染系统功能，是针对渲染过程的增强，深入了解渲染管线的设计思想，请查看
  元编程渲染管线。
- <LinkTo kind="拓展-component-自定义组件--page">自定义组件 (Component)</LinkTo>: 在页面和编辑器的组件选择中，增加自定义组件。

快速了解拓展开发，请点击查看下篇 <LinkTo kind="拓展-快速开始--page">快速开始</LinkTo>。
