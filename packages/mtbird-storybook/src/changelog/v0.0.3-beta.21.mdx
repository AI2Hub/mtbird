# 0.0.3-beta.21

本版本继续优化编辑体验

## 新功能

### 增加对其面板

针对绝对定位组件，增加对其面板，可以针对组件快速操作。如选择一个组件，对其则是跟其父组件对其。如选择多个组件，则是互相间对其

### 顶部快速增加按钮组支持拖拽添加

在顶部快速增加按钮，可以支持拖拽添加组件

### 文本组件支持编辑

文本组件可以双击直接编辑，同时也支持在富文本编辑器中修改样式

### 支持折叠菜单组件

支持容器组件折叠菜单和折叠菜单面板

### 列表组件支持拉取表单数据

SaaS 的表单数据模块，改为由 精卫渲染器 实现，内部包含一个表格组件，拉取表单数据

## 修复已知问题

1. 修复 PC 端页面渲染问题
2. 修复颜色组件透明度无法修改问题
